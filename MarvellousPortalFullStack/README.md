# MarvellousPortalFullStack

A Spring Boot backend application that manages batch data 
using REST APIs and MongoDB.

## 🚀 Features
- CRUD operations on batch records
- RESTful API endpoints
- MongoDB integration

## 🛠️ Tech Stack
- Java
- Spring Boot
- MongoDB
- REST APIs

## 📌 API Endpoints
| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | /batches | Get all batches |
| GET | /batches/{id} | Get batch by ID |
| POST | /batches | Create new batch |
| PUT | /batches/{id} | Update batch |
| DELETE | /batches/{id} | Delete batch |

## ⚙️ How to Run
1. Clone the repository
2. Make sure MongoDB is running
3. Open in IntelliJ IDEA
4. Run `MarvellousPortalApplication.java`
5. API runs on `http://localhost:8080`
